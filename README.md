# Whats Clone

A Dummy WhatsApp Clone based on this design: [WhatsApp Redesign CONCEPT](https://www.behance.net/gallery/72309913/WhatsApp-Redesign-CONCEPT)

## Features

- List chats by pinned and unpinned ones
- Take Pictures
- Record Videos
- Record Audio
